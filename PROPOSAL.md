# Brian-PersBudgApp-Proposal

1. Project Idea 

The Personal Budget Application helps users track their income, expenses, and savings in one place. It exists to give individuals better visibility into their spending habits and help them make informed financial decisions through budgeting and expense tracking. 

2. Features a

Users will be able to: 

| -Create a personal account and securely log in.                          |
| -Add income transactions.                                                |
| -Add expense transactions.                                               |
| -Categorize expenses (Food, Transportation, Entertainment, Bills, etc.). |
| -Edit or delete transactions.                                            |
| -Create monthly budgets for spending categories.                         |
| -View budget progress and remaining balances.                            |
| -Track total income, expenses, and savings.                              |
| -View transaction history.                                               |
| -Filter transactions by category and date.                               |
| -View charts and visual reports of spending habits.                      |
| -Receive alerts when approaching or exceeding budget limits.             |
| -Store data securely for future access.                                  |

 

3. Target Audience 

This application is designed for students, young professionals, and individuals who want a simple way to manage their personal finances. Many people struggle to understand where their money goes each month, and this application provides an easy-to-use solution for tracking spending, creating budgets, and building better financial habits without requiring advanced financial knowledge.
